✅ Notion MCP — pronto para uso

Instalado neste projeto: artethan.ai

Verificação real feita: claude mcp get notionApi retornou Status: ✔ Connected. Está funcionando.

📍 Onde usar

Só funciona dentro desta pasta (artethan.ai), porque foi instalado com escopo local. Se abrir o Claude Code em outro projeto seu, o Notion não vai aparecer lá — só aqui.

🔄 Precisa reiniciar?

Se esta janela do Claude Code já estava aberta antes da instalação, feche e abra o Claude Code de novo nesta pasta para o servidor aparecer na sessão.

🧪 Primeira coisa para testar

Digite no chat do Claude Code, dentro deste projeto:

Busque no meu Notion por páginas e me liste os títulos que encontrar.

O esperado: o Claude vai usar a ferramenta de busca do Notion e te devolver uma lista de páginas/bancos de dados que sua integração tem acesso. Se vier lista vazia, é sinal de que nenhuma página foi conectada à integração ainda (volta no Notion e conecta as páginas que quer usar).

🛠️ Comandos úteis

  • claude mcp list — lista todos os servidores MCP conectados
  • claude mcp get notionApi — mostra status detalhado deste servidor
  • /mcp — dentro de uma sessão do Claude Code, mostra os MCPs ativos
  • claude mcp remove notionApi -s local — remove esta instalação

⚙️ O que o Claude pode fazer agora

  • Ler e editar páginas do Notion (em Markdown)
  • Consultar e atualizar bancos de dados
  • Criar páginas e bancos de dados novos
  • Buscar conteúdo no seu workspace
  • Ler/criar comentários, ver usuários

Lembrete: sua integração só acessa as páginas que você conectou manualmente a ela no Notion. O resto do workspace fica fora do alcance do Claude.
